# AbdulatifOS

A playful browser-based personal web operating system built with HTML, CSS, and JavaScript.

## Files

- `index.html` — main desktop UI and app shell
- `styles.css` — visual styling for desktop, windows, icons, and apps
- `script.js` — app templates, window handling, clock, and theme logic

## Features

- Boot screen that transitions into a desktop
- Top bar with live date and time
- App icons for About, Projects, Notes, Terminal, Gallery, Music, and Settings
- Window system with draggable, closable app windows
- Gallery app with image cards

## Usage

1. Open `index.html` in a browser.
2. Click **Enter desktop** to load the OS.
3. Click an app icon to open its window.

## Notes

- The gallery now includes live image cards from online sources.
- The terminal app supports a few demo commands like `help`, `clear`, `echo`, `date`, and `projects`.
